בדיקות תקינות (Checklist להגשה)
A) ייחוד שמות

Client1: name = A

Client2: name = A
✅ מצופה: השני מקבל Name already exists וממשיך לבקש שם עד שמצליח.

Client1 עושה QUIT/נסגר חלון

Client2 שוב מנסה A
✅ מצופה: עכשיו מותר.

B) Lobby / CREATE / JOIN

A עושה CREATE 2

B עושה LIST ורואה את המשחק WAITING

B עושה JOIN <id>
✅ מצופה: START נשלח לשניהם, תור X מתחיל.

מקרה קצה:
4. C עושה JOIN <id> למשחק של 2
✅ מצופה: ERR full.

C) Turn logic

כשזה לא תורך:

נסי להקליד 00 / 0 0
✅ מצופה: ERR “Not your turn… only INFO/LEAVE/QUIT”

ואז מיד INFO
✅ עובד

ואז LEAVE
✅ עובד, המשחק נסגר לכולם וחוזרים ללובי.

כשזה כן תורך:

0 0 עובד ומעדכן לכולם.

D) LEAVE סוגר משחק לכולם (גם לא בתור)

משחק RUNNING

שחקן שלא בתור עושה LEAVE
✅ מצופה: השני מקבל END וחוזר ללובי.

E) ניתוק פתאומי (הבאג החדש שטיפלת)

משחק RUNNING

אחד עושה Ctrl+C / סוגר חלון
✅ מצופה: השחקנים האחרים מקבלים END וחוזרים ללובי.

ורצוי לבדוק בשני מצבים:

ניתוק בזמן WAITING (לפני שמתחיל)

ניתוק בזמן RUNNING (באמצע משחק)
